Ayuda y Tutorial

¿Cómo funciona esta visualización?

Esta herramienta simula el funcionamiento de un data center en tiempo real, mostrando:

  • Flujo de datos entre servidores
  • Uso de recursos (CPU, RAM, red)
  • Temperatura y consumo energético
  • Conexiones de red y seguridad
Nota: Esta es una simulación interactiva, pero puede conectarse a sistemas reales mediante API.

Controles principales

Añadir servidor:
Crea un nuevo servidor
Añadir datos:
Inyecta paquetes de datos
Procesar datos:
Envía datos a la salida
Configuración:
Ajusta velocidad, tamaño, etc.

Consejos

Ráfaga de datos: Simula picos de tráfico con el botón correspondiente.
Refrigeración: Mantén activado el enfriamiento para evitar sobrecalentamiento.
Conexiones: Los servidores se conectan automáticamente cuando están cerca.

Conexión a sistemas reales

Para conectar con un data center real:

  1. Configura el endpoint de la API
  2. Establece las credenciales de acceso
  3. Selecciona los parámetros a monitorizar
